# Instalação da Extensão

Este documento fornece as instruções detalhadas para instalar a extensão no Google Chrome.

## Como utilizar

Para utilizar a extensão, siga os passos abaixo:

### Google Chrome

1. Abra o Chrome e acesse o endereço: `chrome://extensions/`

2. Ative o **Modo de Desenvolvedor** (no canto superior direito da página).

3. Clique em **Carregar sem compactação**.

4. Selecione a pasta onde os arquivos da extensão foram extraídos.

5. A extensão estará instalada e pronta para uso.
